# practica-6-esp32-buzzer-mario-pwm
Esta es la práctica del ESP32-C3 con buzzer pasivo mediante PWM, donde se reproduce una versión simplificada de la melodía de Super Mario Bros. En el documento se indica que el buzzer se conecta al GPIO1 o GPIO2 y a GND, y que el archivo puede guardarse como main.py en el ESP32-C3
